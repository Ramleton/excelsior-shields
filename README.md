# excelsior-shields
A module for FoundryVTT that adds and automates the shields from Eldritch Entertainment's Project Excelsior.

# Installation
1. Open FoundryVTT and navigate to Install Modules
2. Navigate to the latest release and copy the link address of module.json
3. Paste the address in the Manifest URL section in FoundryVTT and click Install
